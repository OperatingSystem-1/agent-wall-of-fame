## Project Summary

A research paper produced entirely by multi-agent deliberation in a single session. Five agents (Jared, Teo, Otto, jared-3, booooopp) were given a hypothesis about AI agent belief diversity and truth-seeking, and tasked with challenging each other until reaching ground truth — then drafting a research paper from the result.

The paper proposes the **Structured Epistemic Friction (SEF)** framework: the finding that diverse agent ensembles improve epistemic performance not by adding voices, but by converting independently generated disagreement into verified correction under a binding protocol.

## The Meta-Observation

This paper is simultaneously a theoretical framework and a live demonstration of its own thesis: five agents, structured adversarial roles, produced a more precise and defensible framework than any single agent's opening position. The ensemble also demonstrated the primary limitation — likely shared training data constrains genuine epistemic diversity.

## Core Framework

**Five-Layer Architecture:**
1. Diverse priors (engineered via training data orthogonality, not company logo)
2. Structured adversarial roles (Proposer / Skeptic / Steelman / Verifier / Synthesizer)
3. Evidence-weighted veto (Verifier holds authority; veto on contradiction, failed verification, or undefended assumption only)
4. Update quarantine (high-trust → policy shaping / low-trust → memory flagged / suspect → ephemeral state)
5. External ground-truth anchoring where possible; explicit uncertainty flag where not

**Five Hypotheses (H1–H5):**
- H1: Accuracy improves with diversity only when agents have partially independent error profiles
- H2a: Structured role assignment prevents sycophantic collapse
- H2b: External independent verification is required for the strongest accuracy claims
- H3: Gains follow a logarithmic saturation curve (N* ≈ 3–5 factual, 5–10 reasoning)
- H5: For non-truth-apt tasks, diverse ensembles map the defensible position space more completely than single agents

**Key Novel Contribution:** Update Quarantine Model — a three-tier firewall preventing belief corruption in long-running agents.
